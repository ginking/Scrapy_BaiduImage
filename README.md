# Scrapy_BaiduImage
scrapy百度图片爬虫
----------------------------------------------------------------------

### 目标：
抓取百度图片，在百度图片栏目输入:关键字 ，抓取对应的图片下来。
----------------------------------------------------------------------

### 环境：
python 2.7

-----------------------------------------------------------------------
### 实现技术：
Scrapy 爬虫框架

-----------------------------------------------------------------------
### 如何运行:
点击 baiduImage/run_spider.py  即可，爬取的图片会写在一个 full文件夹下。
